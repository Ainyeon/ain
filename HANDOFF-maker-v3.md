# HANDOFF — maker-v3 (사용자 피드백 13건: 커스텀 확장 + 시니어 친화)

브랜치: `feature/maker-v3` (main 미병합, feature/maker-v2 위에 쌓음).
대원칙 준수: 완성된 템플릿 + 넓어진 편집 영역, 전부 단계값 조절이라 **망가뜨릴 수 없는 편집**.

## ① 파일과 역할

## ② 테스트 결과 (전부 통과, Playwright 자동 검증)

- **T1**: 첫 상태 = 빈 검색창 + 범용 문구 미리보기 정상. 지역 입력 → 캔버스 해시 변화(실제 렌더 확인). 업종 검색·직접 입력 정상
- **색**: 그라데이션 디폴트(에인연 블루→버건디) 시작, 단색↔그라 전환·프리셋 원터치 모두 캔버스 반영
- **배경 슬라이더**: 0/50/100 각각 해시 상이, 중간 명도(50)에서 본문 흰색 유지(픽셀 검증 228+/255) — 전 구간 가독성 확보, 100(블랙)은 대표색 그라데이션 푸터로 자동 전환
- **크기**: 드롭다운 12종 전수 내보내기 정상(각 해상도 픽셀 일치: 1080×1080 ~ 3508×2480), 직접 입력 1200×1800 정상, 범위 밖(9999) 안내 문구+무시. 명함(1063×591) 같은 초저높이 규격도 자동 축약 레이아웃
- **탭 선택 조절**: 제목/내용/상호줄/하단 클릭 → 패널 표시, 크기·간격·정렬 각각 해시 반영, 요소별·전체 리셋 정상, localStorage `adj` 저장 확인, 선택 테두리는 미리보기 전용(해제 시 원본 해시 복귀 = 내보내기 무오염)
- **T2 레이아웃 6종**: 상호 해시 전부 상이(6종 각각 고유 렌더), 배너=사진 숨김+문구·대표색 노출, 사진+문구=슬롯 1개, 이름표 편집·프리셋 반영, 배너 문구 탭 조절 동작
- **시니어**: 44px 미만 조작 요소 0건(375px 스캔), 아이콘 단독 버튼 0건(위로/아래로/삭제 텍스트화), 첫 방문 3단계 안내 1회 표시→"다시 보지 않기"/"시작하기" 후 재표시 없음
- **개발자 문구**: `실시간|즉시 반영|디바운스|100ms|베타|BETA` grep — 사용자 노출 0건 (홈·허브·메이커 3페이지)
- **375px 가로 스크롤**: 3페이지 모두 0
- **PC/폰 픽셀 동일**: 375px/1280px 뷰포트 동일 상태 해시 일치
- **Lighthouse** (gzip 서버·headless): 허브 **96/100**, notice **96/100**, compare **95/100** (Perf/A11y). CLS 0.476→0.005 (동적 컨트롤 자리 예약 + 캔버스 즉시 첫 페인트)
- 샘플: `docs/maker-v3-samples/` — T1 그라데이션 화이트/그레이50/블랙, 명함·직접입력 크기, T2 레이아웃 6종 각 1장
