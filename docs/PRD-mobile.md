# PRD — 에인연 모바일 퍼스트 재구축

브랜치: `feature/mobile-rebuild` · main 배포 금지 · 2026-07-04

## 1. 배경 / 사용자

- 사용자: HVAC·인테리어 **현장 종사자**. 70%+ 모바일 접속 가정.
- 사용 환경: 야외·이동 중·장갑 낀 손. **한 손 조작, 엄지 도달 범위** 중심.
- 핵심 여정: 출근길 3분 안에 "오늘 시세 → 이번 주 입주 → 새 공고"를 확인하고 닫는다.

## 2. 모바일 UX 원칙

1. **엄지 우선**: 핵심 내비게이션은 화면 하단 고정 탭바. 상단 nav는 보조.
2. **터치 타겟 ≥ 44×44px**: 탭바·버튼·칩·카드 링크 전부. 인접 타겟 간격 ≥ 8px.
3. **본문 ≥ 16px**: 야외 시인성. 보조 텍스트도 13px 미만 금지.
4. **모션 경량화**: 마우스 파랙스·3D 틸트는 모바일(coarse pointer) 비활성 (기존 가드 유지).
   스크롤 리빌은 유지하되 blur 강도 축소(8px→4px) — 저사양 기기 페인트 비용 절감.
5. **오프라인 내성**: 현장 음영지역 대비 — 마지막 브리핑·시세 1회분은 비행기모드에서도 열람.
6. **데스크톱 무회귀**: ≥900px 화면은 기존 motionsites 스타일 그대로. 탭바는 모바일 전용.

## 4. 완료 기준 (Definition of Done)

- [ ] 375px 뷰포트에서 전 페이지 가로 스크롤 0 (scrollX 이동 불가로 검증)
- [ ] 탭바·주요 버튼 터치 타겟 44×44px 이상 (측정 스크립트로 검증)
- [ ] 본문 텍스트 16px 이상 (콘텐츠 영역 기준)
- [ ] 모바일에서 파랙스·틸트 미동작, 리빌 동작
- [ ] Pretendard 서브셋 woff2 ≤ 900KB, font-display: swap
- [ ] 전 페이지 canonical·og:url·og:title·og:description·og:image(절대 URL) 완비
- [ ] manifest.json + 아이콘(192/512) + 서비스워커 → 설치 가능(Lighthouse installable)
- [ ] 비행기모드(오프라인)에서 브리핑·시세 페이지가 마지막 캐시본으로 열림
- [ ] Lighthouse 모바일: Performance ≥ 90, Accessibility ≥ 90 (로컬 서버 측정)
- [ ] 데스크톱(1440px) 기존 렌더 회귀 없음

## 5. 비범위 (이번 작업에서 안 함)
